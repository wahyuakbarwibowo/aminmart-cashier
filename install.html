Berikan izin akses Lokasi untuk aplikasi RawBT agar dapat mendeteksi printer Bluetooth.

5

Lakukan test print di RawBT untuk memastikan printer berjalan dengan baik sebelum menggunakan Aminmart Cashier.

8

Mulai Menggunakan

Buka Aminmart Cashier dan mulai kelola bisnis Anda. Untuk cetak struk, pastikan RawBT sudah berjalan dan printer terhubung.
